# Agent Warm Up

Do not write any documentation files. Reply only in chat.

Do not start any work. Present your plan and wait for approval before beginning implementation.

## Overview

Instructs agents to present their plan and wait for approval before starting implementation, emphasizing no documentation and chat-only communication.
