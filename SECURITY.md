# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email **hello@birdhouselabs.ai** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes if you have them

We appreciate responsible disclosure and will credit researchers who report valid vulnerabilities.

## Scope

Birdhouse is a local-first developer tool. It is not designed to be exposed to the public internet without additional security controls. Running Birdhouse on a publicly accessible server without a firewall or authentication layer is outside the supported configuration.
