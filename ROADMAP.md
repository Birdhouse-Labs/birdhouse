# Birdhouse Roadmap

The roadmap lives in [GitHub Discussions → Roadmap](https://github.com/Birdhouse-Labs/birdhouse/discussions/categories/roadmap).

Each post describes an upcoming initiative and the thinking behind it. Upvote what you want to see prioritized. Comment to share feedback.

## Suggest a Feature

Open an [Idea in GitHub Discussions](https://github.com/Birdhouse-Labs/birdhouse/discussions/new?category=ideas). Describe the problem you're trying to solve, not just the solution. Features with broad community support and clear alignment with the [roadmap](https://github.com/Birdhouse-Labs/birdhouse/discussions/categories/roadmap) get prioritized.
